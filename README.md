# 🌊 Splash — High-Fidelity Water Reclamation Systems

Splash is a state-of-the-art web application showcasing next-generation, chemical-free molecular water engineering, sustainable municipal treatment facilities, and advanced industrial purification solutions.

Designed with a sleek, immersive dark mode aesthetic, Splash utilizes modern typography, premium animations, and interactive elements to deliver an elite digital presentation of modern ecological engineering.

---

## 🚀 Key Visual & UX Features

* **Cinematic Video Hero**: Implements a parallax-tracked background utilizing custom frame-buffered boomerang video loops for a smooth, high-fidelity landing experience.
* **Curved 3D Cylinder Deck**: An interactive, custom-curved 3D scrolling catalog showcasing industrial solution sectors (Municipal, Heavy Industry, Ecological Biota, and Quantum Membranes) with real-time kinetic SCADA-style canvas visualizers.
* **Kinetic Text Fade**: A scroll-triggered, letter-by-letter typographic opacity transition for the core brand vision statement.
* **Refined Design Tokens**: Premium glassmorphic navigation layout, curated earth-tone borders, and micro-interactions optimized for high-end presentation devices.
* **Turnkey Consultation Hub**: An interactive specification portal allowing clients to outline system capacities and download engineering schematics.

## 🛠️ Technology Stack

* **Frontend Library**: React 19 (TypeScript)
* **Build Tool**: Vite (Ultra-fast HMR)
* **Styling**: TailwindCSS & Custom Vanilla CSS Variables
* **Animations**: GSAP (Mouse Parallax) & Framer Motion / Motion (Kinetic scroll timelines & canvas loops)
* **Icons**: Lucide React

---

## 💻 Run Locally

### Prerequisites
* **Node.js** (v18+ recommended)
* **npm** or **yarn**

### Installation & Startup
1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
